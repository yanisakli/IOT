## Features
- Alarm clock :
    -   Tazer after 1 minute late.
- Sleep monitoring :
    -   Record sounds during the night.
    -   Detect movement during sleep.
- Connected speaker
- Record voice messages
- Light + soft word detection according to sleep brightness/cycle
